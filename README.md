# Rock-Paper-scissor-game
This is my first django project 

Rock Paper Scissor Game (Django)

A simple web-based Rock Paper Scissor game built using Django (Python).
This project allows a user to play against the computer and keeps track of scores using Django sessions.

🚀 Features
Play Rock, Paper, Scissor against computer
Random computer moves
Score tracking (User vs Machine)
Session-based score storage
Reset game option (Quit)
🛠️ Technologies Used
Python
Django
HTML
inline css
